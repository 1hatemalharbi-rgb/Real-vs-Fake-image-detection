# Real vs Fake Image Detection

## How to run

1. Download the required files from Google Drive:
   - Dataset (archive.zip):  
     https://drive.google.com/file/d/1PWMJuFqfEXGuGW1kTU9QZZdtmv2CxOXK/view?usp=drive_link
   - Fine-tuned model (finetune_resnet18.pth):  
     https://drive.google.com/file/d/1iKdqmKkNHtq0JlbCrpGBy-W4xYhij50o/view?usp=drive_link
   - Frozen baseline model (worse_freeze_resnet18.pth):  
     https://drive.google.com/file/d/1PCQea07rm9r35xiMxE3w8j7wyx2p016Q/view?usp=drive_link

2. Upload all downloaded files to **Google Colab**.

3. Open the notebook and run the cells in order.
